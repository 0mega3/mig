    Sidebar

    The sidebar displays a list of rooms with the number of users and a list of users in the current room.

    Example:

    [Rooms]
#general (2)

[Users]
John
Alice

    Message area

    The messages area displays the user's messages and service information.

    Status line

    The status line shows important information about the connection.

    Format: user | #room | Status: status

    • user - Current user
    • #room - Current #room

    • status - Disconnected | Connected | Inactive

    Example:

    Status: Disconnected
Alice | #general | Status: Connected
Alice | [1 room] | Status: Inactive

    Command line

    The command mode is activated by pressing the (:) key (autofocus).

    Command Reference

    Basic:

    :[c]onnect {username}<Enter>    Connect the user to the room #general
:[j]oin {#room}<Enter>          Join the current user to the room #room
:[p]art [{#room}]<Enter>        Leave the current room or room #room
:[s]end {message}<Enter>        Send a message
:[q]uit<Enter>                  Disconnects the user from all rooms

    The part of the command in square brackets :[q]uit means that the basic command have short synonym :q.

    Service:

    :rooms<Enter>                   Shows all the user's rooms
